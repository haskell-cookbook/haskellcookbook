About

Haskell cookbook is a site for sharing haskell code snippets, in the same vein as Python recipes.

The site is forked from django snippets and uses a Foundation based theme for the frontend. The social icons used in login page are the courtesy of Icondock.

EpicEditor, Pygments and CodeMirror are the plugins used to add and edit code. Haskellcookbook is proudly hosted at Openshift.
